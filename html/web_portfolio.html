This project was completed during my internship at the Jamaica Teaching Council government agency. I was tasked with rebuilding their intranet on the wordpress CMS platform to improve task management, communication and workflow efficiency of the employees. The intranet has been completed with features such as a notice board, a google calendar integrated event calendar, a messaging system, a ticket based help desk, a staff directory, document repository and an inventory management system tied to a Caspio database. The look and feel of the website was created from scratch using css and functionality was added to many sections of the website using javascript and php.

ATTY

html

css

javascript

This project was the final, end-of-semester assignment for the collaboration between my classes “Web Management” and “Usability and the Online User Experience”. We were tasked with locating a real client with a web presence and improving that presence via their website and/or social media, as well as the user experience for their customers. For my client I went with the governmental body National Land Agency and primarily focused on their website. I conducted research and collected data with surveys, interviews with the client and users, conducted user experiments, and reviewed their analytics, all with the intention to locate the pain points that the users are experiencing. Recognizing that the main issue that users were experiencing was finding information and accessing their services, I went ahead with my redesign for NLA's website as well as some of their services like document tracking and title searching.
